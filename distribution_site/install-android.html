Cara Instal APK

Langkah instalasi

  1. Download APK sesuai arsitektur HP Anda dari halaman utama.
  2. Buka file APK dari notifikasi download atau File Manager.
  3. Jika diminta, aktifkan izin "Install unknown apps" untuk browser/file manager.
  4. Lanjutkan instalasi sampai selesai.
  5. Buka aplikasi dan login seperti biasa.
Tips: Pilih APK `arm64-v8a` untuk mayoritas HP Android terbaru.
