Cara Update Aplikasi

  1. Saat notifikasi update muncul di aplikasi, klik "Update Sekarang".
  2. Download APK sesuai arsitektur device Anda.
  3. Install APK baru, data lama tetap aman (tidak perlu uninstall).
  4. Jika update bertipe wajib (major), aplikasi lama akan diblok sampai update selesai.
  5. Untuk update patch/minor, Anda bisa pilih "Nanti".
Jika download gagal, buka lagi halaman utama distribusi dan unduh manual dari tombol Android.
